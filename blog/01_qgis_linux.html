¿Qué es PoP Os?

Pop Os es una distribución de Linux basado en Ubuntu, desarrollado el año 2017 por System 76, empresa que actualmente apuesta por el uso del software libre y se dedicada a dar soporte a sus usuarios, fabricar ordenadores y servicios.

Características de PopOs: * Es una distribuición de linux super ligera (2GB de RAM) al igual de Xubuntu. * La instalación es rápida y sencilla * Podemos descargar una ISO que instala automaticamente los drives de Nvidia, si en caso se cuenta con uno. * Cuenta con una “Tienda de aplicaciones” que posee una variedad de softwares para ingeniería, bioinformática, Deep Learning y para la creación de contenido. * Por ultimo la seguridad y la privacidad son los pilares fundamentales en Pop Os, este cuenta con un cifrado de disco que nos permite proteger nuestra información.

Existen más características, si quieres obtener más información, te estaré adjuntando en la parte final algunas referencias que pueden ser de tu interés.

¿Qué es QGIS?

QGIS es un software de Sistema de Información Geográfica libre y código abierto orientando al análisis,edición,creación, visualización y publicación de datos espaciales. Este en los últimos años ha tenido una gran popularidad y acogida por muchos usuarios, principalmente por la gran comunidad y la variedad de complementos o Plugins(actualmente existe un total de 829) que están en constantemente desarrollo.

Características de QGIS:

  • Presenta una interfaz gráfica amigable.
  • Corre en múltiples sistemas operativos (GNU/Linux, Unix, Mac OS, Microsoft Windows y Android.
  • Soporta numerosos formatos vectoriales, raster , bases de datos y servicios web.
  • Integra de mejor manera otro freeware como SAGA, GRASS ,R entre otros.
  • Presenta una comunidad muy dinámica.

Existen 3 tipos de versiones en QGIS, estas son las siguientes:

  1. LTR: Lanzamiento a largo plazo (Modificado 1 vez al año)
  2. LR: Última versión (Modificado cada 4 meses)
  3. DEV: Versión de prueba (Se modifica en meses, semanas o días)

En esta oportunidad vamos a descargar la versión de desarrollo, esto con la finalidad de explorar las nuevas funcionalidades que estarán disponible en los próximos lanzamientos.

Instalación de la version DEV de QGIS

Para poder instalar cualquier versión de QGIS, te recomiendo visitar la misma página oficial, ya que este nos ofrece una guía amplia paso a paso de como instalar qgis en difirentes sistemas operativos.

Es importante saber identificar la versión de la distribución de linux con la cual estás trabajando, esto lo puedes hacer directamente desde la shell tipieando el siguiente comando:

antonybarja@pop-os:~$  cat /etc/os-release
NAME="Pop!_OS"
VERSION="20.10"
ID=pop
ID_LIKE="ubuntu debian"
PRETTY_NAME="Pop!_OS 20.10"
VERSION_ID="20.10"
HOME_URL="https://pop.system76.com"
SUPPORT_URL="https://support.system76.com"
BUG_REPORT_URL="https://github.com/pop-os/pop/issues"
PRIVACY_POLICY_URL="https://system76.com/privacy"
VERSION_CODENAME=groovy
UBUNTU_CODENAME=groovy
LOGO=distributor-logo-pop-os

Como pueden notar nosotros estamos trabajando con la distribución PopOS y tenemos la version groovy

Entonces para poder instalar es necesario ver el siguiente cuadro de especificaciones:

De cuardo al cuadro de específicaciones procederemos a tipear los siguientes códigos en nuestro /etc/apt/sources.list:

antonybarja@pop-os:~$ sudo gedit /etc/apt/sources.list

Esto nos abrirá un archivo con nuestro editor de texto y solo pegamos los siguientes comandos:

## QGIS 
deb [arch=amd64] https://qgis.org/ubuntu-nightly groovy main
deb-src https://qgis.org/ubuntu-nightly groovy main

Luego le damos Ok y guardamos, para finalizar la instalación empleamos los siguientes comandos

sudo apt update && sudo apt upgrade
sudo apt install qgis qgis-plugin-grass
Y listo ahora podemos explorar los nuevos features de QGIS
